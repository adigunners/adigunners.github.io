Testing Instructions

  1. Open index.html and winners.html in separate browser tabs
  2. Use browser developer tools to test these key breakpoints:
    • 320px - Small mobile
    • 375px - iPhone SE
    • 768px - Tablet
    • 1024px - Desktop
    • 1440px - Large desktop
  3. Take screenshots at each breakpoint for both pages
  4. Save screenshots in: testing/visual-regression/baseline/
  5. Naming convention: [page]-[width]px.png
